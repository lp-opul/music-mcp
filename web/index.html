Add Distro to Claude Desktop

1 Clone and build 
git clone https://github.com/lp-opul/Distro.git
cd Distro
npm install && npm run build
2 Add to Claude Desktop config

Open or create: ~/Library/Application Support/Claude/claude_desktop_config.json

{
  "mcpServers": {
    "distro-mcp": {
      "command": "node",
      "args": ["/path/to/Distro/dist/client.js"]
    }
  }
}

Replace /path/to/Distro with where you cloned it (e.g. ~/Distro)

3 Restart Claude Desktop
Quit and reopen Claude Desktop. You'll see the Distro tools available in the tools menu.
Note: This gives Claude direct access to music distribution tools. You can ask Claude to create artists, generate music, upload tracks, and submit releases to streaming platforms.
